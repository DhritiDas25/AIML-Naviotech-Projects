# AIML-Naviotech-Projects
This repository contains AIML Projects.
The projects are as followed:
Project 1: Medical Insurance Price Prediction
Type: Regression
Objective: Predict the annual premium cost for a customer based on individual health and demographic factors.
Key Features: Age, BMI, smoking status, region, and number of children.
Model Used: Linear Regression or Random Forest Regressor.
Project 2: Big Mart Sales Prediction
Type: Regression
Objective: Forecast sales for various products across different store outlets to help the retailer understand which items drive the most revenue.
Key Features: Item weight, fat content, visibility, store type, and store establishment year.
Model Used: XGBoost or Gradient Boosting Regressor.
Project 3: Cancer Classification (Neural Networks)
Type: Deep Learning (Binary Classification)
Objective: Classify tumors as malignant or benign using a Multi-Layer Perceptron (MLP).
Key Features: Cell nuclei characteristics (radius, texture, perimeter, area, etc.).
Architecture: Input layer, multiple hidden dense layers with ReLU activation, and a Sigmoid output layer.
Project 4: Movie Recommendation System
Type: Information Retrieval / Filtering
Objective: Recommend movies to a user based on their interests or the similarity between films.
Methodology: Content-Based Filtering using Cosine Similarity on metadata like genres, keywords, cast, and director.
Tools: TfidfVectorizer and CountVectorizer.
Project 5: Diabetes Prediction
Type: Classification
Objective: Predict whether a patient has diabetes based on diagnostic measurements.
Key Features: Glucose level, Blood Pressure, Insulin, BMI, and Age.
Model Used: Support Vector Machine (SVM) or Logistic Regression.
